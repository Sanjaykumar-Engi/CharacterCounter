📌 Character Counter Web App

A simple and interactive real-time character counter web application built using HTML, CSS, and JavaScript.
It tracks user input, shows live character count, remaining characters, includes a progress bar.

🚀 Features
📝 Real-time character counter
📊 Live progress bar with color change
⚠️ Character limit warning (200 max)
💾 Auto-save using localStorage
🔄 Restores text after page refresh
📱 Fully responsive design
🎨 Clean and professional UI

🛠️ Technologies Used
HTML5
CSS3 
JavaScript

⚙️ How It Works
      User types in the textarea
      JavaScript listens to input event
      Character count updates in real-time
      Progress bar fills dynamically
